Simple Export to PNG

Ru:

Быстрый экспорт в PNG (Tools -> Scripts -> Export to PNG) экспортирует файл в текущую директорию без диалога выбора имени файла/директории (XXX.kra -> XXX.png)

Горячая клавиша F11. Чтобы изменить ее, откройте в текстовом редакторе файл simpleExportPNG.action (в папке "actions") и измените в нем строку <shortcut>f11</shortcut> (например: <shortcut>ctrl+alt+shift+f11</shortcut>) или вовсе удалите файл, если для этой функции вам вообще не нужна горячая клавиша

En:

The Simple Export to PNG Plugin (Tools -> Scripts -> Export to PNG) export you file to the current directory as png without saved dialog (XXX.kra -> XXX.png)

Current keyboard shortcut F11. To change it edit simpleExportPNG.action file (in "actions" directory) and change line <shortcut>f11</shortcut> (for example: <shortcut>ctrl+alt+shift+f11</shortcut>) or remove this file if you do not need keyboard shortcuts for this
